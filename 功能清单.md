## 🎯 使用流程

1. **首次访问** → 显示登录页面
2. **点击注册** → 填写邮箱、用户名、密码
3. **自动定位** → 获取国家位置
4. **注册成功** → 自动登录
5. **查看地球** → 看到自己的位置标记
6. **上传头像** → 点击头像进入设置
7. **邀请朋友** → 分享网站链接
8. **查看统计** → 底部显示用户数和国家数
