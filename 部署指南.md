# 🚀 Vercel一键部署指南

## 方法1：使用GitHub（推荐）

### 步骤1：上传到GitHub

1. 访问 https://github.com
2. 点击右上角 "+" → "New repository"
3. 填写仓库名称：`globe-users`
4. 选择 "Public"
5. 点击 "Create repository"

6. 在你的电脑上打开命令行，进入项目文件夹：
```bash
cd "c:\Users\22546\OneDrive\Desktop\New folder"
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/globe-users.git
git push -u origin main
```

### 步骤2：部署到Vercel

1. 访问 https://vercel.com
2. 点击 "Sign Up" → 选择 "Continue with GitHub"
3. 授权Vercel访问你的GitHub
4. 点击 "New Project"
5. 找到 `globe-users` 仓库，点击 "Import"
6. 直接点击 "Deploy"（不需要配置任何东西！）
7. 等待1-2分钟
8. 看到庆祝动画 🎉
9. 点击 "Visit" 访问你的网站！

## 方法3：使用Vercel网页上传

1. 访问 https://vercel.com
2. 登录账号
3. 点击 "Add New..." → "Project"
4. 选择 "Deploy from a Git repository" 或 "Import from template"
5. 如果没有Git仓库，可以直接拖拽文件夹到页面
6. 点击 "Deploy"
7. 完成！

## ❓ 常见问题

**Q: 需要付费吗？**
A: 不需要！Vercel免费版完全够用。

**Q: 需要配置数据库吗？**
A: 不需要！所有数据存储在浏览器本地。

**Q: 需要环境变量吗？**
A: 不需要！这是纯前端项目。

**Q: 可以自定义域名吗？**
A: 可以！在Vercel项目设置中添加自定义域名。

**Q: 数据会丢失吗？**
A: 数据存储在用户浏览器中，清除浏览器数据会丢失。

**Q: 支持多少用户？**
A: 理论上无限，但每个用户的数据只在自己浏览器中。
